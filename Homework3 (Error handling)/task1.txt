Користувач надіслав порожній JSON -> 400 Bad Request
Користувач намагається видалити товар, якого не існує -> 404 Not Found
У базі даних стався збій під час запиту -> 500 Internal Server Error
Клієнт робить 100 запитів на секунду при ліміті 10 -> 429 Too Many Requests
Користувач не надав Bearer-токен -> 401 Unauthorized
